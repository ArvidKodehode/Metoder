Del 1

1. Hva er forskjellen mellom en metode og en funksjon?

En funksjon er en selvstendig blokk med kode som kan kalles uavhengig. En metode er en funksjon som er bundet til et objekt.


// Eksempel på en funksjon
function minFunksjon() {
    console.log("Dette er en funksjon.");
}
minFunksjon();

// Eksempel på en metode
const objekt = {
    metode: function() {
        console.log("Dette er en metode.");
    }
};
objekt.metode();

2. Hva er en static-metode?

En static-metode er definert på klassenivå og kan kalles direkte på klassen uten å opprette en instans.


class Klasse {
    static hils() {
        console.log("Hei fra en static-metode!");
    }
}
Klasse.hils();

3. Hva er en instance-metode?

En instance-metode er bundet til en spesifikk instans av en klasse og opererer på dens data.


class Klasse {
    hils() {
        console.log("Hei fra en instance-metode!");
    }
}
const instans = new Klasse();
instans.hils();

Del 2


    
1. Lag et array som inneholder 20 tall

    
For å lage et array med 20 tall, kan vi bruke Array.from med en tilfeldig genereringsfunksjon.

    

const tallArray = Array.from({ length: 20 }, () => Math.floor(Math.random() * 100));
console.log(tallArray);

    


    
2. Sjekk om det inneholder det samme tallet flere ganger

    
For å sjekke om et array inneholder duplikater, kan vi bruke some-metoden sammen med indexOf.

    

const harDuplikater = tallArray.some((val, idx) => tallArray.indexOf(val) !== idx);
console.log(harDuplikater ? "Arrayet inneholder duplikater." : "Arrayet inneholder ingen duplikater.");

    


    
3. Hvis ikke, erstatt 4 av tallene med det samme tallet

    
Hvis det ikke er duplikater, kan vi erstatte fire tall med samme verdi.

    

if (!harDuplikater) {
    const nyttTall = 99; // Et eksempel på et fast tall
    tallArray[0] = nyttTall;
    tallArray[1] = nyttTall;
    tallArray[2] = nyttTall;
    tallArray[3] = nyttTall;
}
console.log("Oppdatert array:", tallArray);

    


    
4. Sorter disse tallene i stigende rekkefølge

    
For å sortere tallene i stigende rekkefølge, kan vi bruke sort-metoden med en sammenligningsfunksjon.

    

tallArray.sort((a, b) => a - b);
console.log("Sortert array:", tallArray);

    


    
5. Gang alle tallene dine med 3 og console.log ut det nye arrayet

    
For å multiplisere alle tallene med 3, kan vi bruke map-metoden.

    

const gangetArray = tallArray.map(num => num * 3);
console.log("Array ganget med 3:", gangetArray);
